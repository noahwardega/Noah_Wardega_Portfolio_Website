GameHub

Overview

GameHub is a browser-based web application that brings together three classic games—Wordle, Rock Paper Scissors, and Tic Tac Toe—into one unified platform. Designed for quick, casual gameplay, each game is built using React components and delivers an interactive, responsive user experience. Users can navigate seamlessly between games, enjoy intuitive interfaces, and engage in simple, replayable challenges.

Team/Roles

Noah Wardega: Developer

Solution

Technology Used

GameHub is built using a modern front-end development stack focused on interactivity and responsiveness. Key technologies include:

  • React.js – for building the user interface with reusable, component-based architecture.
  • JavaScript – for implementing game logic, interactivity, and dynamic behavior.
  • HTML & CSS – for structuring content and styling the visual layout of the application.
  • React Hooks – for managing component state and side effects in a clean and functional way.
  • React Router – for navigating between different games and sections of the hub
  • VS Code – as the primary development environment
  • Git & GitHub – for version control and project hosting.
