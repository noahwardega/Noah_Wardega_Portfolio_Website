PlanIt

Overview

PlanIt simplifies event management by allowing users to create, organize, and RSVP to events effortlessly. From the home page, users can view upcoming events they've RSVP’d to and delve into event details with a simple click. Creating events is intuitive, with options to set event specifics and invite guests directly through the app. PlanIt also provides visibility into guest RSVP statuses, ensuring seamless coordination and anticipation of attendees.

Solution

Technology Used

PlanIt is built using a modern web development stack. Key technologies include:

  • Java & Spring Boot – for building the backend, handling APIs, and managing business logic.
  • Thymeleaf – for rendering dynamic HTML templates.
  • Spring Data JPA – for database interaction using object-relational mapping.
  • MySQL – as the primary relational database to store user, event, and RSVP data.
  • HTML & CSS – for structuring and styling the frontend interface.
  • Maven – for managing project dependencies and build configuration.
  • JUnit & Spring Boot Test – for writing and running unit and integration tests.
  • Git & GitHub – for version control, collaboration, and project hosting.
  • IntelliJ – as the primary development environment
