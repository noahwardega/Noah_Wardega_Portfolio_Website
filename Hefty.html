Hefty

Overview

Hefty is a mobile app design prototype for a modern banking platform. It allows users to view account balances and transaction history, trade cryptocurrencies, send and receive money, and manage bills directly through the app. The prototype includes key screens such as Home, Exchange, Deposit, Transfer, Bills, Settings, and Support, offering a streamlined and user-friendly experience tailored for digital financial management.

Team/Roles

Noah Wardega: UI Designer
Peter McCartney: Flow Designer

Solution

Technology Used

Hefty was designed using these tools:

  • Trello – for Project Management, allowing tbe team to organize models, files, and tasks
  • Figma – for Designing and Prototyping the User Interface, enabling collaboration and visualizing the app’s layout and functionality
